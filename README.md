# AnimatedAuth

## Description
AnimatedAuth is a React Native frontend mobile app that demonstrates a smooth animated authentication flow with modern UI.

## Features
- Login & Register screens
- Smooth animations
- Clean UI design
- React Native frontend only

## Tech Stack
- React Native
- Expo
- JavaScript
